# 互動小說模式使用說明書

本文件說明如何在 EVA「小說互動模式」中使用管理指令，動態切換敘事風格與張力規則。

---
## 一、指令格式
- 所有管理指令皆以前綴 **`#cmd`** 開頭，僅供系統解析，不會送給 GPT。
- 指令區分為兩類：`preset` 與 `rule`。

### 1. Preset 指令
| 指令 | 功能 |
|------|------|
| `#cmd preset add <預設包名>` | 載入指定預設包，啟用其中所有規則 |
| `#cmd preset remove <預設包名>` | 卸載預設包 |
| `#cmd preset list` | 列出目前啟用中的預設包 |

> 範例：
> ```
> #cmd preset add dark_knight_pack
> ```
>
> 作用：啟用「黑暗騎士」風格，包含道德灰階、混沌敘事等規則。

### 2. Rule 指令
| 指令 | 功能 |
|------|------|
| `#cmd rule add <規則名> key=value ...` | 動態新增單一規則（可帶參數） |
| `#cmd rule set <規則名> key=value ...` | 修改現有規則參數 |
| `#cmd rule remove <規則名>` | 停用指定規則 |
| `#cmd rule list` | 查看目前生效規則 |
| `#cmd rule reload` | 重新載入 `story_config/runtime` 與預設包，立刻生效，不需重啟 |

> 範例：
> ```
> #cmd rule add humor_tone level=0.9
> ```
>
> 作用：提高整體幽默感。

---
## 二、常用預設包
| 預設包 | 描述 |
|--------|------|
| `dark_knight_pack` | 黑暗寫實、道德灰階、心理博弈強 |
| `konosuba_comedy` | 輕鬆吐槽、反套路喜劇 |
| `foreshadow_drama` | 多伏筆鋪陳，重情感反轉 |

---
## 三、互動流程建議
1. **啟動模式** → 選擇「小說互動模式」。
2. **載入預設包**（可選）：
   ```
   #cmd preset add konosuba_comedy
   ```
3. **開始遊戲**：照畫面提示輸入選項或 `t` 自訂動作。
4. **動態調整**：途中想增加張力，可輸入：
   ```
   #cmd rule add conflict_question
   ```
5. **查看現況**：
   ```
   #cmd rule list
   #cmd preset list
   ```
6. **存檔離開**：在指令列輸入 `s` 以寫檔保存。

---
## 四、術語說明
- **Rule**：一段程式邏輯，負責改寫 Prompt 或更新世界狀態。
- **Preset**：Rule 的組合包，以 YAML / RON 檔描述。
- **WorldState**：記錄角色信任度、謊言、情感旗標等資訊的資料結構。

---
## 五、後續擴充
若想新增自己風格：
1. 在 `story_config/presets/` 建立 `my_style.ron`：
   ```ron
   [
       "humor_tone(level=0.8)",
       "moral_dilemma()",
   ]
   ```
2. 進入遊戲輸入：
   ```
   #cmd preset add my_style
   ```
即可套用。
